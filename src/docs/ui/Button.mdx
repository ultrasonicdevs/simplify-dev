# Button

The `Button` component is a versatile UI element that supports different sizes, variants, and types. It can render as either a native `<button>` or an `<a>` element, depending on the `as` prop.

<Canvas of={ButtonStories.ButtonDefault} />
<Controls of={ButtonStories.ButtonDefault} />


## Props
<ArgTypes of={ButtonStories} />

## Basic Usage

Below is an example of how to use the `Button` component directly:

```tsx
import { Button } from '@ui';

export const Example = () => (
  <Button variant="primary" size="md">
    Click me
  </Button>
);
```
